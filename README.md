# 就労継続支援B型作業所 ホームページ

## プロジェクトの目的

このプロジェクトは、**就労継続支援B型作業所のホームページ制作を通じて、利用者がコーディングやプログラミングを学習すること**を目的としています。

実際のWebサイト制作を通じて、HTML、CSS、JavaScriptなどのWeb技術を実践的に学び、ITスキルの習得を目指します。

## ブランチ運用

このリポジトリでは、以下の3つのブランチで開発を進めています。

### 1. `main` ブランチ
- **用途**: 本番環境のソースコード
- **書き込み権限**: プロジェクト管理者（野村さん）
- **説明**: 最終的な成果物が反映されるメインブランチです

### 2. ClaudeCode用作業ブランチ
- **用途**: AI（ClaudeCode）による開発・コーディング支援
- **命名規則**: `claude/` で始まるブランチ（例: `claude/add-contact-section-xxxxx`）
- **説明**: ClaudeCodeが自動生成するコードの作業領域です

### 3. `work/hoshi` ブランチ
- **用途**: 利用者さんの学習・開発用ブランチ
- **説明**: 利用者さんが実際にコードを書いて学習するための作業ブランチです

## 開発の流れ

1. 各作業ブランチで開発を進める
2. プルリクエスト（PR）を作成してレビュー
3. レビュー後、`main` ブランチにマージ
4. `main` ブランチの内容が本番環境に反映される
